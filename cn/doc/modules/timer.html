timer

timer 定时器 (WeexSDK 0.7+)

Weex 封装了一系列 API，以便在固定的延迟后启动或停止某个一次性或重复的任务。

注意事项：该模块只保证尽力交付，而并不能提供精确的延迟功能。如果相应的线程处于繁忙状态，实际延迟可能超过用户希望的延迟。

事实上，这个模块是一个针对 HTML5 Timer API 的 polyfill，开发者应该使用 HTML5 Timer API，而不应该直接使用该模块。

示例

<template>
  <div>
    <text>setTimeout timeout=1000</text>
    <div style="
        height: 200px; width: 750px;
        background-color: {{bg1}};
        justify-content: center;
        align-items: center
      " onclick="{{timeout}}">
      <text style="font-size: 60px; color: #ffffff">Click</text>
    </div>
    <text style="margin-top: 100px">setInterval interval=1000</text>
    <div style="
        height: 200px; width: 750px;
        background-color: {{bg2}};
        justify-content: center;
        align-items: center
    " onclick="{{interval}}">
      <text style="font-size: 60px; color: #ffffff">{{txt}}</text>
    </div>
  </div>
</template>

<script>
  var fnId = undefined;
  var isOnRepeat = false;
  module.exports = {
    data: {
      bg1: 'yellow',
      bg2: 'red',
      txt: 'Click to start'
    },
    methods: {
      timeout: function () {
        var self = this;
        setTimeout(function () {
          self.bg1 = self.bg1 == 'green' ? 'yellow' : 'green'
        }, 1000);
      },
      interval: function () {
        var self = this;
        if (!isOnRepeat) {
          fnId = setInterval(function () {
            self.bg2 = self.bg2 == 'red' ? 'green' : 'red'
          }, 1000);
          self.txt = 'Click to stop';
          isOnRepeat = true;
        } else {
          clearInterval(fnId);
          self.txt = 'Click to start';
          isOnRepeat = false;
        }
      }
    }
  }
</script>

在线体验

API (不推荐直接通过 module API 使用)

该模块中的所有超时或间隔时间都以毫秒为单位。

并且，所有时间都必须是一个非负整数（允许的最大值是0x7FFFFFFF）。如果一个时间值被设置为负数，那么它将被重置为0。例如，一个延迟为负数的任务将被立即放入任务执行队列中。

setTimeout(fn, timeout)

setTimeout() 函数用于在指定的毫秒数后调用一个目标函数，可以使用 clearTimeout() 函数在目标函数被调用前来取消调用。目标函数只会被执行一次，如果你需要重复执行，请使用 setInterval() 函数。

参数

  • fn (function): 将要被执行的目标函数
  • timeout (number): 指定目前函数前延迟的毫秒数

返回值

代表目标函数的一个数字 id 值，可以使用这个 id 值和 clearTimeout() 函数来取消定时器。

setInterval(fn, interval)

setInterval() 函数会在一个指定的时间间隔之后调用一个目标函数，并且该目标函数会以相同的时间间隔被重复调用，直到 clearInterval() 函数被调用。

参数

  • fn (function): 被调用的目标函数
  • interval (number): 重复调用目标函数的间隔时间

返回值

代表目标函数的一个数字 id 值，可以使用这个 id 值和 clearInterval() 函数来取消定时器。

clearTimeout(fnId)

clearTimeout() 函数被用来清除一个通过 setTimeout() 设置的定时器。如果目标函数还没有被执行，那么调用该函数后将取消执行目标函数，否则，调用将不会有任何效果。

参数

  • fnId (number): setTimeout() 函数返回的 id 值

clearInterval(fnId)

clearInterval() 被用于清除通过 setInterval() 函数设置的定时器。

参数

  • fnId (number): setInterval() 函数的返回值
